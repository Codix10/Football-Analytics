# 📊 La Liga 2024–25: xG vs Actual Goals Analysis

## 🔍 Overview
This project explores one of the most interesting questions in football analytics:

**Who were the most clinical finishers in La Liga last season?**

Using Expected Goals (xG) vs Actual Goals, this analysis highlights which players outperformed their chances and which ones underperformed.

---

## 📈 Visualization

The scatter plot compares:
- **Expected Goals (xG)** on one axis  
- **Actual Goals** on the other  

**Interpretation:**
- Points **above the diagonal line** → Players scoring more than expected (clinical finishers)  
- Points **below the line** → Players underperforming their chances
